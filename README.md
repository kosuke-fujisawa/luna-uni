# 魔法少女部ルナユニオン ―星に託す襷―

> 世代交代型・魔法少女育成シミュレーション

## 一言で

「栄冠ナイン」を下敷きにした、世代交代型・魔法少女育成シミュレーション。
プレイヤーは魔法が使えない30代の顧問教師となり、唯一の魔法高校で少女たちを育て、見送り、次の世代へ襷を繋いでいく。
