# 戦闘システム・信頼値

## 基本構造

「ポケモン × 栄冠ナイン」

| 項目 | 内容 |
|------|------|
| 編成 | 3人 |
| 形式 | ターン制 |
| 行動順 | 素早さ順 |
| 操作 | 半オート |

プレイヤーは細かい操作をしない。指示するのは「方針」。

---

## 方針指示

実際のスキル選択はキャラが自動で行う。プレイヤーが指示できるのは以下の方針のみ。

- 攻撃重視
- 防御重視
- 温存
- 支援優先
- 一点突破

---

## 信頼値

栄冠ナイン的要素。各キャラに「信頼値」が存在し、指示UIで表示される。

### 信頼が高い場合
- 指示成功率上昇
- コンボ発生
- 自主的好判断

### 信頼が低い場合
- 暴走
- 空回り
- 指示無視

信頼値は育成・イベント・日常のやり取りを通じて変化する。
